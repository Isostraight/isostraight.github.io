KittyBot

KittyBot V1.2 Release

Features

  • Enhanced BallCam Control
  • Force Jump (Simple and Double)
  • Customizable Key Binds
  • Optimized Kickoff Strategies
  • And More!

How to Use

  • BallCam is set to "space" by default. To change, edit the settings.py file.
  • Force Jump Simple is "u". Modify this in bot.py.
  • Force Jump Double is "i". Change this key in bot.py.

How to Set Up

  • Step 1: Download the required files by clicking the button below.
  • Step 2: Install Python (3.9 or later) and pip.
  • Step 3: Run pip install -r requirements.txt to install dependencies.
  • Step 4: Launch Rocket League and run the script main.py.
  • Step 5: Customize settings in settings.py for optimal gameplay.
